# RSL-plus 1.1.3 — обновление диагностики и навигации

Архив накладывается поверх актуального `master` с заменой файлов. Номер версии остаётся `1.1.3`.

## Основное

- Problems очищается после закрытия макрофайла.
- Новые проверки: `DEBUGBREAK`, неиспользуемые объявления и известные Import, повторный `ELSE`, использование до объявления, неоднозначные ссылки.
- Все группы диагностик управляются из настроек VS Code.
- Quick Fixes для безопасно исправляемых сообщений.
- `Shift+F12` ищет ссылки по проиндексированному проекту.
- Semantic Tokens уточняют подсветку Macro, Class, Method, параметров, переменных и констант.
- Старые root dev-зависимости удалены или обновлены.
